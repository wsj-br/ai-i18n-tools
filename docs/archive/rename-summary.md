# Historical note: rename to ai-i18n-tools

The npm package was renamed from `@transrewrt/i18n-tools` to **`ai-i18n-tools`** and the CLI binary from `i18n-tools` to **`ai-i18n-tools`** to avoid namespace conflicts on npm.

- **Default config file**: `ai-i18n-tools.config.json` (see [GETTING_STARTED.md](../GETTING_STARTED.md)).
- Current install: `npm install ai-i18n-tools` / `pnpm add ai-i18n-tools`.

This file is kept for changelog context only; it is not required reading for new users.
